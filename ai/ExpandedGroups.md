# План реализации раскрытого вида группы (Expanded Group View)

Этот документ описывает шаги по внедрению детального экрана группы в `GroupsScreen.kt` согласно предоставленному макету.

## 1. Состояние и Навигация (State Management)
- [ ] В `GroupsScreen` добавить переменную `selectedGroupId` через `rememberSaveable { mutableStateOf<String?>(null) }`.
- [ ] Реализовать логику переключения:
    - Если `selectedGroupId == null` — показывать список всех групп.
    - Если `selectedGroupId != null` — показывать `ExpandedGroupContent`.
- [ ] В `GroupItemCard(onClick = { ... })` установить ID нажатой группы.

## 2. Основной контейнер `ExpandedGroupContent`
- [ ] Создать Composable функцию, принимающую объект группы и `onBack: () -> Unit`.
- [ ] Использовать `LazyColumn` как основу для всего экрана (для поддержки скролла).
- [ ] Контейнер должен иметь фон `MaterialTheme.colorScheme.onSurface` и скругления сверху (`RoundedCornerShape`).

## 3. Секция 1: Header (Информация о группе)
- [ ] **Верхняя панель:** `Row` с кнопкой закрытия (иконка стрелки вниз `keyboard_arrow_down`) слева и кнопкой настроек (`settings`) справа.
- [ ] **Центральный блок:** `Column` с `Alignment.CenterHorizontally`.
    - Аватар группы: круглый `AsyncImage` или `Icon`.
    - Название: `Text` (fontFamily = `myTypeFamily`, FontWeight(900), размер ~24.sp).
    - Подзаголовок: "X members" (серый цвет).

## 4. Секция 2: Missed Alarms (Пропустили будильник)
- [ ] Заголовок: Текст "MISSED ALARMS" красного цвета (`Color(0xFFE57373)`), капсом.
- [ ] Контейнер: `Column` с внутренними отступами и скругленными углами.
- [ ] **MissedAlarmItem (Карточка):**
    - Фон: Светло-красный градиент или сплошной цвет.
    - Лево: Аватар пользователя + Имя.
    - Право: Время пропущенного будильника + Зеленая кнопка вызова (иконка `phone`).

## 5. Секция 3: Upcoming Alarms (Ближайшие будильники)
- [ ] Заголовок: Текст "UPCOMING ALARMS" серого цвета.
- [ ] Контейнер: Светло-серый фон с закруглением.
- [ ] **UpcomingAlarmItem (Карточка):**
    - Список из первых 3 пользователей, отсортированных по времени.
    - Элементы списка разделены небольшим отступом.
    - Контент: Аватар, Имя, Время будильника.

## 6. Анимация и Стилизация
- [ ] Обернуть смену состояний (список -> детали) в `AnimatedContent` для плавного перехода.
- [ ] Убедиться, что `AddGroupButton` скрывается, когда группа раскрыта.
- [ ] Использовать `MaterialTheme.colorScheme.tertiary` для основного текста, как в `ProfileScreen.kt`.
